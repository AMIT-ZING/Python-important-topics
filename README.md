# Python-important-topics

## ITERATOR :
An iterator is a state so that it remembers where it is during iteration  

## ITERABLE :
Iterables are something that can be looped over  
### e.g.
A list can be called iterable as we can loop over a list  

```python
list = [1, 2, 4, 6, 7]

for i in list:
    print(i)
```
